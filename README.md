# RxJava Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![RxJava](https://img.shields.io/badge/RxJava-3.0-blueviolet.svg)](https://github.com/ReactiveX/RxJava)

Тестовое приложение для изучения RxJava в Android. Демонстрирует работу с Observable, Single, Flowable и управление подписками.

## Функциональность

- Создание различных типов Observable (just, delay)
- Управление потоками с Schedulers
- Использование CompositeDisposable для отписки
- Работа с Single для асинхронных операций
- Обработка подписок в UI потоке

## Технологии

- Kotlin
- RxJava 3
- CompositeDisposable
- AndroidSchedulers
- Schedulers
